# CyberTruck-Website


Tesla Cybertruck 3D Landing Page

A modern landing page showcasing the Tesla Cybertruck with a 3D interactive model and advanced animations.

Features

- 3D Cybertruck Model: Interact with the model using Three.js.
- Smooth Animations: Scroll and hover effects powered by GSAP.
- Responsive Design: Optimized for all screen sizes.

Technologies Used

- HTML5, CSS3, JavaScript
- Three.jsfor 3D rendering
- GSAP for animations

Getting Started

1. Clone the repository.
2. Open the `CyberTruck.html` file in your browser.
3. Add the 3D model (`cybertruck.glb`) in the same directory or provide a hosted URL.
